Documentação

A documentação vai te ajudar a melhor entender o funcionamento da nossa API e garantir que você possa aproveitar o máximo possível dela.

GET

localhost:3000/Api/livros

O método GET, utilizado para desenvolver uma sequência de consulta, é o responsável por exibir os livros registrados em nossa API. Dessa forma não é obrigatório ser feito uma passagem de parâmetros para que esse método possa ser executado com sucesso. Mas, ele irá listar todos os livros que estiverem registrados. Para buscar um livro específico, o ID de cada obra pode ser usado como parâmetro.

Usando parâmetros

Para utilizar esse parâmetro, basta adicionar a "/" e o número do ID que deseja buscar. A fim de exemplificar esse método, vamos utilizar o ID “2” como parâmetro.

Exemplo: localhost:3000/Api/livros/2.
